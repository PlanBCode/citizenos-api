Was passiert als Nächstes?

1

 Die Moderatoren werden den Bericht prüfen. Wenn sie der gemeldeten Verletzung zustimmen, wird der Zugriff auf das Thema beschränkt (Benutzer erhalten eine Warnung, bevor sie das Thema anzeigen).

2

 Inzwischen werden Administratoren und Bearbeiter des Themas aufgefordert, das Thema zu bearbeiten, um die Verletzungen zu lösen. Die Moderatoren können dann das Thema prüfen und Beschränkungen entfernen, sobald aller Verletzungen gelöst sind.

3

 Wir werden Sie in den folgenden Fällen benachrichtigen:
  • Moderatoren wenden Beschränkungen für das Thema an
  • alle Verletzungen wurden gelöst und Beschränkungen entfernt.

Thema anzeigen

Sie können den Beitrag unverändert lassen, aber Moderatoren können darüber entscheiden, ob er standardmäßig ausgeblendet werden soll (Benutzer erhalten eine Warnung, bevor sie das Thema sehen).
