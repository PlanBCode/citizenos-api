New Feature: Idea Gathering
Citizen OS has launched the first version of a new feature – idea gathering! This feature is designed to make it easier for users to engage diverse audiences and gather ideas for their civil society initiatives.
The idea gathering feature enables users to:
  • Post ideas
  • Read and reply to each other’s ideas
  • Like and save ideas to personal favourites
  • Admins can add ideas to folders to aid in analysing and sorting the gathered input
  • Start a voting phase for selected ideas
The new idea gathering feature
Visit the platform!
The idea gathering feature supports a wide range of needs, from gathering ideas for public space improvements and policy changes to planning strategies or democracy festivals.
Users can post, discuss, and like ideas, while admins manage engagement, set up the vote, and report on actions taken as a result.
These features help to improve community engagement through a dynamic, multi-way dialogue, going beyond simple surveys or one-way communications.
